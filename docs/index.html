Dungeon Adventure RPG

A visually immersive, turn-based RPG dungeon crawler built with Python and Pygame.

Download Main GitHub Repo

Quest Features

⚔️

6 Playable Classes

Choose your hero: Warrior, Mage, Rogue, Paladin, Ranger, or Monk. Each path features unique stats and special passives.

🎲

Tactical Combat

Experience deep, turn-based battles. Utilize upper, middle, and lower attacks alongside critical hits and combos.

🗺️

Procedural Paths

Explore an ever-changing overworld. Traverse rooms filled with enemies, merchants, traps, and ancient shrines.

💰

Treasure & Trade

Amass gold to trade with dungeon merchants. Stock up on vital potions or wield the legendary Royal Claymore.

💾

Persistent Progress

The dungeon never forgets. Use the integrated save/load system to continue your adventure whenever duty calls.

🕹️

Graphical Evolution

A journey from text-based CLI to a fully immersive Pygame graphical interface with custom UI elements.

Development Status

Active Development (BETA)

The game is undergoing continuous refinement. While currently in an incomplete development stage, it offers a distinct and playable RPG experience. Community feedback is vital to our journey!

Evolutionary Roadmap

Revision 1: The Foundation Pure CLI logic and text-based dungeon crawling.
Revision 2: Atmosphere Added original audio tracks and refined class passives.
Revision 3: The Vision (Current) Full graphical overhaul using Pygame with immersive UI.
